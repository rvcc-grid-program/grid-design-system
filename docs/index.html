GRID design system

The design system for GRID course content: one set of Console tokens and components flowing from wiki markdown to a styled web page, a Canvas LMS fragment, and print. This page is built by the pipeline it documents — everything you’re looking at is the system rendering its own markdown.

Learning objectives

When you have completed this page you will be able to:

  • Recognize the Console theme shared with the GRID claim app
  • Know where the specimen and the Canvas fragment live
  • Find the spec documents in the repository
!Warning

Canvas strips box-shadow, letter-spacing, opacity, negative margins, and text-transform. Every component here is designed to survive that — borders carry structure, shadows are garnish, and uppercase is baked in at build time.
